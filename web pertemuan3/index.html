Jenis-Jenis CSS Position

by Dyra

Position: Static

- Static adalah nilai default dari properti position di CSS.

- Jika suatu elemen tidak diberi position, maka secara otomatis posisinya adalah static.

- Elemen dengan position static akan mengikuti alur normal dokumen (normal flow) artinya elemen ditampilkan di tempat sesuai urutan HTML.

CIRI-CIRI STATIC :

路 Tidak terpengaruh oleh properti top, right, bottom, atau left. 路 Posisi elemen tidak bisa digeser pakai properti-properti di atas.

Position: Relative

- Relative artinya posisi elemen relatif terhadap posisi normalnya di dokumen.

- Jika top, left, right, atau bottom, maka elemen akan bergeser dari posisi aslinya, tapi ruang kosong di tempat awalnya tetap ada.

CIRI-CIRI RELATIVE :

- Tetap mengikuti alur normal dokumen, jadi space-nya di layout tidak hilang.

- Bisa digeser pakai top, left, right, bottom.

- Sering dipakai untuk membuat patokan bagi elemen absolute di dalamnya (jadi parent-nya).

Position: Absolute

- absolute artinya elemen keluar dari alur normal dokumen (tidak mempengaruhi elemen lain).

- Elemen akan diposisikan relatif ke parent terdekat yang punya position: relative, absolute, atau fixed.

- Kalau tidak ada parent yang di-position, maka elemen absolute nempel ke atau viewport.

CIRI-CIRI ABSOLUTE :

- Bisa dikontrol pakai top, left, right, bottom.

- Tidak meninggalkan ruang di tempat aslinya (lepas dari flow).

- Sangat cocok untuk bikin layout yang presisi atau overlay.

Position: Fixed

- fixed artinya elemen lepas total dari alur dokumen dan nempel ke layar (viewport), Posisi elemen tidak terpengaruh parent selalu mengacu ke layar, dan Saat discroll, elemen tetap diam di tempatnya, seolah-olah "mengambang".

CIRI-CIRI FIXED :

- Bisa pakai top, left, right, bottom. dan tidak terpengaruh parent sama sekali. Tidak meninggalkan ruang di tempat aslinya (lepas dari flow).Cocok untuk navbar yang selalu di atas, floating button, popup, banner iklan.

Position: Sticky

- Sticky adalah posisi gabungan antara relative dan fixed.

- Awalnya elemen berada di posisi normalnya, tapi akan "menempel" di posisi tertentu (misalnya di atas layar) saat halaman discroll.

- Dia hanya nempel selama masih dalam area parent-nya.

CIRI-CIRI STICKY :

- Bisa dikontrol pakai top, left, right, bottom (paling sering pakai top).

- Elemen tidak langsung nempel, tapi nempel saat discroll dan hanya sampai batas parent-nya habis.

- Cocok untuk bikin header section, navigasi, atau judul tabel yang tetap di atas saat scroll.
